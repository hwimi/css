자바스크립트의 파라미터 특징 /기능

  • 파라미터의 개수와 관계없이 함수(메서드) 이름이 같다면 호출가능
  • 파라미터의 개수가 불일치하여 받지 못하는 값이 있다면 undifined 처리
  • 값을 받아오지 못하는 파라미터가 undifined 처리가 되는걸 막기위해 기본값 설정을 할 수 있음. =>optional parmter
  • 모든 파라미터는 aregument라는 프로퍼티를 이용하여 파라미터를 관리 aregument라는 배열로 담가셔 관리
  • optional parmter는 저장 안됨
  • optional parmter는 다른 파라미터보다 뒤에 잇어야한다.(가장 끝에 존재해야한다)
  • rest paramete:남는(이외의, 남는 것을) 파라미터
  • 사용할 파라미터는 일반파라미터로 배정하고 나머지 파라미터들을 통합하여 하나의 이름으로 배정하는 기법
  • ...:spred 연산자라고 하여, 배열이나 객체를 복제하는데도 사용함.
