string:가장많이 다루는 데이터 타입

  • java에서 String method와 거의 유사함
  • 문자열 데이터는 index(0부터 시작)을 가지고있음
  • 문자열 값 내에서 (br) 개행 하는 것은 추천하지 않음.
  • new String():객체 타입으로 선언가능/추천하지않음
  • length:문자열의 총길이
  • 추출관련 메서드:substring(),substar(),slice()
  • 문자대체관련 메서드 :replace(old,new):처음 찾은값만 간으
  • trim():양쪽공백제거
  • padStart(길이,채울문자)/padEnd(길이,채울문자) 0100100-4******/***시
  • charAt(index)/indexOF()
  • starwidth("문자열")/endwith("문자열") 해당 문자열 앞/뒤에 문자열이 존재하는 확인하는 메서드라고 return true/fase
